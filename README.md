# Microcontroladores 🤖

Este repositório contém uma coleção de exercícios, exemplos práticos e projetos desenvolvidos durante as aulas da disciplina de **Microcontroladores**. O foco principal é o aprendizado de arquitetura de hardware, manipulação de periféricos e lógica de programação voltada a sistemas embarcados.

## 🛠️ Tecnologias e Ferramentas

* **Linguagem:** C / C++ (Arduino Framework)
* **Plataforma:** Arduino (Uno/Mega/Nano)
* **Simulação:** Wokwi / Proteus / TinkerCAD (opcional)
* **IDE:** Arduino IDE / VS Code + PlatformIO


## 🚀 Como Executar

1.  Certifique-se de ter a **Arduino IDE** instalada.
2.  Clone este repositório:
    ```bash
    git clone [https://github.com/seu-usuario/nome-do-repositorio.git](https://github.com/seu-usuario/nome-do-repositorio.git)
    ```
3.  Abra o arquivo `.ino` desejado.
4.  Conecte seu Arduino ao computador.
5.  Configure a placa e a porta corretas na IDE e clique em **Upload**.

## 📝 Conceitos Estudados

* Manipulação de Registradores.
* Controle de fluxo e lógica condicional em C.
* Interface com sensores e atuadores.
* Gerenciamento de memória e eficiência de código em sistemas limitados.
